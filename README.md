# 🐦 Flappy Bird

A classic **Flappy Bird** clone built with **Unity** and **C#**. Navigate the bird through an endless series of pipes by tapping or clicking — simple to learn, hard to master!

---

## 🎮 Gameplay

- Control a bird that constantly falls due to gravity
- Press **Space** / **Click** / **Tap** to flap and gain height
- Fly through gaps between pipes without hitting them
- Each successful pipe pass increases your score
- The game ends when the bird hits a pipe or the ground

## 🚀 Getting Started

### Prerequisites

- [Unity Hub](https://unity.com/download) installed
- Unity Editor (version **2021.x** or later recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shreyansh109/FlappyBird.git
   ```

2. **Open in Unity**
   - Launch **Unity Hub**
   - Click **"Open"** → Select the cloned `FlappyBird` folder
   - Let Unity import all assets and compile scripts

3. **Run the game**
   - Open the main scene from `Assets/Scenes/`
   - Press the **▶ Play** button in the Unity Editor
